# xkcd comic puller
Silly project that pulls a random comic from [this page](https://xkcd.com/)

## Clone
```console
$ git clone https://github.com/laurenL1202/xkcdpull.git
$ cd xkcdpull
$ pip install -r requirements.txt
```

## Usage
Calling
```console
$ python3 pull.py
```
will save the comic in your system.
